Lecture 25: Introduction to JavaScript & First Interaction

Summary

المحاضرة دي كانت أول خطوة في تعلم JavaScript، اللغة اللي بتخلي صفحات الويب تتحرك وتتفاعل مع المستخدم. عرفنا يعني إيه JS وليه بنستخدمها، واتكلمنا عن أماكن كتابة الكود، إزاي نربط ملف .js بالصفحة، وإزاي نتعامل مع العناصر عن طريق DOM.

Key Points

  1. ما هي JavaScript?

    JavaScript هي لغة برمجة بتشتغل على جانب المستخدم (Client Side) داخل المتصفح، وبتسمحلك تضيف تفاعل للموقع زي الأزرار، التنبيهات، وحساب النتائج. كمان ممكن تُستخدم في مجالات تانية زي:

    • تطوير مواقع الويب (Front-End & Back-End)
    • تطبيقات الموبايل
    • الألعاب
    • تطبيقات سطح المكتب

  2. أماكن كتابة كود JavaScript:

    عندنا كذا طريقة نكتب بيها كود JS:

    • جوه وسم <script> في نفس ملف HTML.
    • في ملف خارجي (زي script.js) وبتربطه في الصفحة عن طريق:
    <script src="js/script.js"></script>

    والأفضل دايمًا نحط الكود في آخر body علشان الصفحة تحمل الأول قبل ما الكود يشتغل.

  3. التعامل مع العناصر باستخدام DOM:

    استخدمنا دالة document.getElementById() علشان نوصل لأي عنصر عن طريق id، زي: 

    var btn = document.getElementById("btn1")

    وبعدين ربطنا حدث (Event) بالزر:

    btn.onclick = function() { div.innerHTML = "islam mohamed" }

    كده لما المستخدم يضغط على الزرار، الكود جوه الـ function بيتنفذ ويغيّر محتوى div.

  4. أنواع الإخراج (Output Methods):

    • document.write() بيطبع نص على الصفحة مباشرة.
    • alert() بيظهر رسالة تنبيه للمستخدم.
    • confirm() بيسأل المستخدم بـ (OK / Cancel).
    • prompt() بياخد إدخال من المستخدم.
    • console.log() بيطبع في Console للمطورين.

  5. تعريف المتغيرات (Variables):

    المتغير هو مكان في الذاكرة بنخزن فيه قيمة. بنستخدم الكلمة var للتعريف، زي:

    var num = 10;

    ملاحظات مهمة:

    • مينفعش نبدأ المتغير برقم.
    • مينفعش نستخدم كلمة محجوزة في اللغة.
    • اختار أسماء تدل على المعنى.

  6. الدوال (Functions):

    اتعلمنا إزاي نكتب دالة ونستدعيها:

    function sum() { console.log(9 + 8); }

    sum(); استدعاء الدالة وتشغيلها.

Notes

  • 💡 استخدم console.log() علشان تتابع نتايج الكود خطوة بخطوة.
  • ⚙️ دايمًا اربط ملف script.js في آخر body عشان الصفحة تحمل الأول.
  • 🚫 متستخدمش أسماء محجوزة زي var أو function كأسماء متغيرات.
  • 🧠 افهم الكود قبل ما تنسخه - التجربة أهم حاجة في تعلم JavaScript.
