Lecture 21: CSS Transform & 2D Transformations

Summary

المحاضرة دي كانت عن خاصية transform في CSS، واللي بنستخدمها لتغيير شكل العناصر وتحريكها أو تدويرها أو تكبيرها أو إمالتها على الصفحة. الخاصية دي قوية جدًا لأنها بتدي تأثيرات بصرية احترافية من غير ما نحتاج أكواد جافاسكريبت. بدأنا بشرح التحويلات الثنائية الأبعاد (2D transforms) زي translate، rotate، scale، وskew، وكمان شفنا الصيغة الرياضية matrix() اللي بتجمعهم كلهم في سطر واحد.

Key Points

  1. transform Property:

    الخاصية الأساسية اللي بنستخدمها لأي تحويل، سواء كان حركة أو تدوير أو تكبير أو إمالة.

    الصيغة العامة:

    transform: function(value);

  2. translate():

    بتحرك العنصر في الاتجاه الأفقي أو الرأسي.

    • transform: translate(200px, 100px); تحريك 200px يمين و100px لأسفل.
    • transform: translateX(200px); تحريك أفقي فقط.
    • transform: translateY(100px); تحريك رأسي فقط.

  3. rotate():

    بتلف العنصر بدرجة معينة حول مركزه.

    مثال:

    transform: rotate(180deg);

    كده العنصر بيلف 180 درجة.

  4. scale():

    بتكبر أو تصغر العنصر على محور X أو Y.

    • transform: scale(1.5, 1.5); تكبير بنسبة 150%.
    • transform: scaleX(2); تمديد أفقي.
    • transform: scaleY(0.5); تقليص رأسي.

  5. skew():

    بتميل العنصر على محور معين علشان تعطي تأثير “الانحدار”.

    • transform: skewX(30deg); ميل أفقي.
    • transform: skewY(20deg); ميل رأسي.

  6. matrix():

    دي طريقة متقدمة بتسمح بجمع أكتر من تحويل (translate, scale, skew) في سطر واحد.

    الشكل العام:

    transform: matrix(scaleX, skewY, skewX, scaleY, translateX, translateY);

    مثال:

    transform: matrix(0.57, 1.13, -0.78, 0.92, 14, -9);

  7. transition + hover Effects:

    استخدمنا transition: all 2s; علشان التحويلات تحصل بسلاسة على مدار ثانيتين.

    ومع :hover طبقنا transform: rotate(180deg); لما الماوس يمر فوق العنصر.

Notes

  • 💡 خاصية transform لا تؤثر على العناصر الأخرى حولها - التغيير بيكون بصري فقط.
  • 🎯 دايمًا استخدم transition علشان الحركة تكون ناعمة وجذابة.

  • ⚙️ ممكن تدمج أكتر من تحويل في سطر واحد:

    transform: rotate(45deg) scale(1.2);
  • 🧠 التحويلات 2D بتتعامل مع المحاور X وY فقط، بينما 3D تضيف العمق (Z) - هنتكلم عنها لاحقًا.
