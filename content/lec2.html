Lecture 2: Introduction to HTML Basics

Summary

المحاضرة دي كانت بتشرح الأساسيات اللي بنبدأ بيها لما نتعلم HTML. اتكلمنا عن فكرة إن HTML هي اللغة اللي بنبني بيها هيكل أي صفحة ويب - يعني بنستخدمها عشان نرتب النصوص والعناوين والصور والأزرار. كمان اتعرفنا على أنواع التاجات (العناصر) اللي بتتكتب ما بين start tag وend tag، وبعض التاجات اللي بتتقفل لوحدها زي <img>. المحاضر وضّح برضه حاجات خفيفة عن الـ SEO والـ CSS كتحضير للمحاضرات الجاية.

Key Points

  1. HTML هي اختصار لـ HyperText Markup Language:

    يعني دي اللغة اللي بنستخدمها عشان نبني الهيكل الأساسي لأي صفحة ويب. بتوضح للمتصفح كل جزء من الصفحة بيعمل إيه، زي العناوين أو الفقرات أو الصور.

  2. العناوين من <h1> لـ <h6>:

    دول بيستخدموا عشان نكتب العناوين داخل الصفحة. <h1> هو العنوان الرئيسي والأهم، وكل ما الرقم يزيد (لحد <h6>) بيكون حجم وأهمية العنوان أقل.

  3. العناصر اللي بتتقفل لوحدها (self-closing tags):

    فيه تاجات مش بتحتاج end tag، زي <img> اللي بنستخدمه لعرض الصور. النوع ده من العناصر بيبقى مكتفي بنفسه.

  4. تكوين العنصر في HTML:

    أي عنصر عادي بيتكوّن من start tag + content + end tag.

    مثال:

    <p>This is the content</p>

  5. مقدمة عن SEO وCSS:

    المحاضر ذكر إن الـ SEO هو تحسين محركات البحث عشان الموقع يظهر بسهولة في جوجل، والـ CSS بتتحكم في شكل الصفحة وألوانها وتصميمها، وهنبدأ فيها قدام.

Notes

  • 💡 خليك دايمًا منظم في كتابة الكود علشان يبقى سهل الفهم بعد كده.
  • ⚙️ افتكر إن HTML بتتعامل مع البنية بس، مش الشكل.
  • 📸 لما تستخدم <img> لازم تضيف خاصية alt علشان الصورة لو ما ظهرتش، يظهر نص بديل.

  • 🧠 ودايمًا علّق على الكود بتاعك عشان تفتكر هو بيعمل إيه:

    <!-- This is a comment in HTML -->
