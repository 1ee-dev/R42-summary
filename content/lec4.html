Lecture 4: CSS Types, Selectors & Lists

Summary

المحاضرة دي كانت عن الطرق المختلفة لكتابة تنسيقات CSS في صفحات الويب، واتكلمنا عن الفرق بين inline وinternal وexternal stylesheets، وكمان عرفنا إزاي نستخدم class وid لتطبيق تنسيقات معينة على عناصر معينة. بعد كده اتعلمنا إزاي نعمل القوائم (مرتبة وغير مرتبة) باستخدام <ul> و<ol>، واتكلمنا عن شوية خصائص زي border وborder-radius اللي بتدي شكل جميل للعناصر.

Key Points

  1. CSS Types:

    فيه 3 طرق نكتب بيهم التنسيقات في CSS:

    /* 1. Inline CSS */
<p style="color: blue; font-size: 16px;">Hello World</p>

/* 2. Internal CSS */
<style>
p {
  color: blue;
  font-size: 16px;
}
</style>

/* 3. External CSS (style.css) */
<link rel="stylesheet" href="style.css">

    حيث:

    • Inline التنسيق بيتكتب جوه التاج نفسه باستخدام خاصية style.
    • Internal بيتكتب جوه صفحة HTML في تاج <style>.
    • External بنربط ملف .css خارجي عن طريق <link>.

  2. Selectors:

    اتكلمنا عن طريقتين أساسيّتين لتحديد العناصر في CSS:

    /* Class selector */
.highlighted {
  background-color: yellow;
  color: black;
}

/* ID selector */
#header {
  font-size: 24px;
  color: blue;
}

    حيث:

    • class بتستخدم مع نقطة (.) للتنسيق على عدة عناصر
    • id بتستخدم مع (#) للتنسيق على عنصر واحد

  3. Buttons Styling:

    اتعلمنا إزاي نغيّر شكل الأزرار باستخدام خصائص مختلفة:

    .button {
  background-color: #4CAF50;
  color: white;
  border: 2px solid #45a049;
  border-radius: 8px;
  padding: 10px 20px;
}

    حيث:

    • background-color لتحديد لون الخلفية
    • color لتحديد لون النص
    • border لتحديد نوع وحجم ولون الإطار
    • border-radius لتدوير الحواف

  4. Lists:

    اتعلمنا نعمل نوعين من القوائم:

    • <ul> - Unordered List قائمة بنقط.
    • <ol> - Ordered List قائمة مرقّمة أو بحروف.

    واستخدمنا خاصية type لتغيير شكل الترقيم في <ol>.

  5. CSS Colors & Backgrounds:

    اتكلمنا عن خاصيتي background-color وcolor، واللي بنستخدمهم لتغيير خلفية العنصر ولون النص. في الأمثلة استخدمنا ألوان زي الأحمر، الأزرق، والأخضر لتطبيق عملي على العناصر المختلفة.

Notes

  • 💡 اشتغل دايمًا باستخدام external CSS عشان التنسيق يكون منفصل وسهل التعديل.
  • ⚙️ استخدم class لو هتكرّر نفس التنسيق على أكتر من عنصر، وid لو العنصر فريد.
  • 🎨 جرّب بنفسك تغيّر في background وborder عشان تشوف التأثير العملي.
  • 🧠 فهمك لأساسيات التنسيق دي هيساعدك تبني صفحات منظمة واحترافية أكتر.
